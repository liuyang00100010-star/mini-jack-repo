# 品牌分析方法论

## 分析流程（6步）

0. **执行摘要** — 核心发现、核心建议、逻辑支撑、预期效果
1. **品牌速写** — 一句话定位、品牌健康度评分卡（知名度/联想/感知质量/忠诚度/溢价能力）
2. **业务类型判断** — A类(消费品牌) / B类(服务型) / C类(本地生意) / D类(B2B) / E类(个人品牌)
3. **市场扫描** — 隐藏前提、竞争格局、行业趋势、客户旅程地图
4. **机会洞察** — 核心机会、TAM/SAM/SOM市场测算、财务影响估算、时光机模型
5. **战略分析** — 商战模型定位、升维打击、品牌架构、定价策略、竞品反应模拟
6. **执行方案与优先级** — KPI框架、30/60/90天里程碑、风险评估与情景分析

## 框架清单（24个）

### 市场与品牌分析（7个）
- 品牌画布 — 品牌全景图
- 品类生命周期 — 判断品类阶段
- SWOT — 内外部优劣势
- 消费者画像 — 目标用户特征
- 需求层次 — 功能/情感/身份需求
- 定位三角 — 品牌-品类-用户三角关系
- 隐藏前提识别 — 行业未被检验的假设（**优先使用**）

### 竞争分析（3个）
- 竞争格局矩阵 — 价格带×人群带定位
- 商战模型 — 进攻/侧翼/游击/防御战
- 护城河 — 竞争壁垒评估

### 产品与定价（2个）
- 品牌架构 — 品牌屋/背书品牌/品牌化屋模式分析
- 定价策略 — 价格锚定/弹性/结构/歧视四层诊断

### 战略与增长（4个）
- 增长飞轮 — 正循环结构
- 价值链 — 全链路价值分布
- JTBD（Jobs to Be Done）— 用户任务理论
- 升维打击 — 跨维度竞争策略

### 创意与内容（3个）
- 品牌语言分析 — 话术体系拆解
- 内容策略 — 平台×内容类型矩阵
- 感官体验 — 五感设计

### 思维模型（5个）
- 多维视角思维 — 多角度审视同一问题
- 局部最优 vs 全局最优 — 避免陷入局部最优陷阱
- 时光机模型 — 发达市场趋势预判中国市场
- 商战模型（战略版）— 竞争定位
- 定位模型 — 心智占位

## 分析过程要求

使用任何框架时，必须展示完整的 4 步分析路径：

1. **框架选择** — 为什么选这个框架？
2. **分析步骤** — 第一步做了什么、第二步发现了什么
3. **推理过程** — 如何从发现推导出洞察
4. **结论** — 最终判断是什么

> 禁止只说"用了XX框架，结论是YY"。必须展开分析路径。
